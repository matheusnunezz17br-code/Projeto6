# Projeto6

# Introdução

Em resumo, consiste numa calculadora simples no qual o usuário tem de digitar dois valores e a operação que deseja fazer.

# Tecnologias Usadas

Python 3.14.

# Como Rodar?

Coloque o código no PyCharm ou qualquer outro programa do mesmo tipo, preencha os valores e a operação e o sistema fará a conta para você.

# O que aprendi?

A usar case de maneira eficiente ao lado de comandos como float e também os operadores.
